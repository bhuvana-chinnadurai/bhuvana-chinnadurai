#### What I'm Building

🏗️ **[Farmaze](https://farmaze.com)** — B2B procurement platform that streamlines how restaurants, cloud kitchens, and food businesses source fresh produce. WhatsApp-first ordering, supplier management, and automated catalog distribution across 12+ microservices.

#### How I Build

I believe in **shipping fast with production-grade quality**. My stack:

- **Backend:** Go, Python (FastAPI), Node.js (Express)
- **Frontend:** React + Vite + Tailwind
- **AI/LLM:** Claude API, tool-use agents, RAG pipelines
- **Infra:** Terraform, Docker, DigitalOcean, Supabase

I write strict TypeScript, type-hinted Python, and idiomatic Go. Every service has proper observability, structured logging, and environment-based config.

**Farmaze** is solving a ₹2 lakh crore+ procurement problem in India's food service industry — fragmented supply chains, zero digitization, WhatsApp-driven chaos turned into structured operations.

**Zeverity** replaces the 3 AM scramble of checking 10 dashboards during an outage with an AI agent that does the investigation for you.
